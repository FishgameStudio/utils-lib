# Security Policy

## Supported Versions
Only the **latest stable version** of this project receives security updates and patches.

## Reporting a Vulnerability
If you discover any security vulnerability, please submit details privately via email:
**[popxhxh@outlook.com](mailto:popxhxh@outlook.com)**

Please include the following information in your report:
- Vulnerability description and reproduction steps
- Affected version scope
- Potential impact and risk level
- Suggested repair solutions (optional)

We will respond to valid security reports as soon as possible, and keep you updated on repair progress.
*(I'm a student)*

## Security Update Statement
We will fix confirmed security flaws in a timely manner, release patched versions and record related changes in release notes.

## Disclosure Policy
Please **do not** publicly disclose undisclosed security vulnerabilities before official fixes are released. We will coordinate proper public disclosure after the problem is resolved.
