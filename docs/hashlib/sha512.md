# Documentation for SHA‑512 Hash Module
Function usage: Implements the standard **SHA‑512 cryptographic hash algorithm**, converts any input string into a fixed 128‑character hexadecimal hash string.


# Documentation of function `rotr64`
Function usage: Performs **64‑bit cyclic right shift** (core bit operation in SHA‑512).
Implementation:
```cpp
constexpr uint64_t rotr64(uint64_t val, int shift) {
    return (val >> shift) | (val << (64 - shift));
}
```
**Logic**: Shift right `shift` bits, move overflowed low bits to the left side.
**Why needed**: SHA‑512 standard requires cyclic rotation for strong bit mixing.


# Documentation of constant `K512`
Function usage: SHA‑512 standard **80 round constants** (sixth roots of first 80 primes).
Definition:
```cpp
inline constexpr std::array<uint64_t, 80> K512 = { ... };
```
**Why needed**: Fixed constants defined by SHA‑512 standard to ensure hash uniqueness and cryptographic security.


# Documentation of internal function `__sha512`
Function usage: Low‑level SHA‑512 core function, processes binary data and outputs a 64‑byte raw hash.
Implementation:
```cpp
inline std::array<uint8_t, 64> __sha512(const std::vector<uint8_t>& data)
```
**Core Steps**:
1. Initialize eight standard 64‑bit hash values `h0 ~ h7`.
2. **Preprocess input**:
   - Append `0x80` padding marker.
   - Fill `0x00` to make total length ≡ 896 mod 1024 bits.
   - Append original 128‑bit big‑endian message length.
3. **Process in 1024‑bit (128‑byte) blocks**:
   - Split block into 16 × 64‑bit big‑endian words.
   - **Expand** to 80 words using `rotr64` and XOR logic.
4. **80 rounds of compression**:
   - Use `s0`, `s1`, `ch`, `maj`, and `K512` constants.
   - Update working variables `a,b,c,d,e,f,g,h`.
5. **Update final hash** state.
6. **Serialize** 8 × 64‑bit hash to 64‑byte big‑endian array.
**Return**: Raw 64‑byte SHA‑512 digest.


# Documentation of function `sha512`
Function usage: Public SHA‑512 interface. Input string → output 128‑char lowercase hex hash.
Implementation:
```cpp
inline std::string sha512(const std::string& plain_text)
```
**Logic**:
1. Convert input string to binary byte buffer.
2. Call core `__sha512` to get raw 64‑byte digest.
3. Convert each byte to two hex characters (0–9, a–f).
4. Return final 128‑character hash string.
**Example**:
`"hello"` → `"9b71d224bd62f350d96d24037de4b39d8c34f714d27c15246d3dcc72fce70e3967fbe8cbb8c6e8215731117b76b0a240bd481c0c8f9dd11a08d915e78955085e"`
